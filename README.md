# todo_list

## Project setup

下载包

```
npm install
```

### Compiles and hot-reloads for development

在根目录打开powershell，输入以下命令，完成运行：

```
npm run serve
```

### Compiles and minifies for production

将vue文件打包为html，css，js文件

```
npm run build
```

### Lints and fixes files

检查与修复文件

```
npm run lint
```
